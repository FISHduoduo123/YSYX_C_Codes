# 第 13 章 本阶段总结
**善于学习的人都应该善于总结。本书的编排顺序充分考虑到知识的前后依赖关系，保证在讲解每个新知识点的时候都只用到前面章节讲过的知识，但正因为如此，很多相互关联的知识点被拆散到多个章节中了。我们一章一章地纵向学习过来之后，应该理出几个横切面，把拆散到各章节中的知识点串起来。请从以下几个方面整理和复习。**

## 一、C的语法规则。

### 1.源文件中所有函数定义之外可以出现哪些语法元素？
- 预处理指令：#include、#define、#ifdef 等  
- 类型定义：typedef、struct、union、enum 的定义  
- 全局变量定义：如 int point_index = 0;  
- 函数声明（原型）：如 void print_map(void);  
- 宏定义：#define MAX_ROW 5  
- 注释：/* */ 或 //  
- 编译指示：#pragma
### 2.函数定义之中可以出现哪些语法元素？
- 变量定义（局部变量）   
- 语句（见下一问）  
- 表达式  
- 注释  
- 其他函数调用  
- 复合语句（块）：{ ... }  
- 空语句：;  
### 3.语句有哪几种？
- 表达式语句	x = y + 1;	  
- 复合语句块	{ int a; a=1; }  
- 空语句	;  
- 条件语句	if (x>0) y=1; else y=0;  
- 选择语句	switch(c) { case 1: ... }  
- 循环语句	while(1) {}、for(i=0;i<10;i++)  
- 控制转移语句	break;、continue;、return;、goto;  
### 4.哪些语法元素需要遵循标识符的命名规则？
**标识符（变量名、函数名、宏名、结构体标签、typedef 名称等）需要遵循命名规则：**
- 变量名：int point_index;
- 函数名：void print_map(void)
- 宏名：#define MAX_ROW 5
- 结构体标签：struct point { ... };
- typedef 名称：typedef int myint;
- 枚举常量：enum { UP, DOWN };
### 5.表达式由哪些语法元素组成？
- 字面量：123、3.14、'a'、"hello"  
- 变量：x、point_index  
- 函数调用：print_map()  
- 算数运算符：+、-、*、/、=、== 等  
- 下标访问：maze[row][col]  
- 成员访问：p.row、p->row  
- 强制类型转换：(int)3.14  
- 条件表达式：a > b ? a : b  
- 逗号表达式：i=1, j=2, k=3  
### 6.到目前为止学过哪些运算符？它们的优先级和结合性是怎样的？
**优先级	运算符	                名称	         结合性	       示例**
```1	  () [] . ->	      函数调用/下标/成员	左 → 右	   func() arr[i] p.x```  
```2	  ++ -- + - ! ~ * &	  单目（后缀/前缀）	    右 → 左	   i++ !flag *ptr```  
```3	  * / %	              乘除取余	           左 → 右	  a * b c % d```  
```4	  + -	              加减	               左 → 右	  a + b c - d```  
```5	  << >>	              移位	               左 → 右    a << 2```  
```6	  < <= > >=	          关系	               左 → 右	  a < b```  
```7	  == !=	              相等性	           左 → 右	  a == b```  
```8	  &	                  按位与	           左 → 右	  a & b```  
```9	  ^	                  按位异或	           左 → 右	  a ^ b```  
```10	  |	                  按位或	           左 → 右	  a | b```  
```11	  &&	              逻辑与	           左 → 右	  a && b```  
```12	  ||	              逻辑或	           左 → 右	  a || b```  
```13	  ?:	              条件	               右 → 左	  a ? b : c```  
```14	  = += -=             赋值	               右 → 左	  a = b```  
```15	  ,	                  逗号	               左 → 右	  i=1, j=2```  
### 7.特殊运算符说明
#### 哪些运算符取操作数的左值？
- 赋值运算符 =：左操作数必须是左值
- 复合赋值 +=、-= 等：左操作数必须是左值
- 自增自减 ++、--：操作数必须是左值
- 取地址 &：操作数必须是左值
- 成员访问 .：左操作数必须是左值
#### 哪些运算符的操作数必须是整型？
- 位运算：& | ^ ~ << >>
- 取余 %
- 自增自减 ++ --
- 移位 << >>
#### 哪些运算符有Side Effect？
- 赋值运算符 = += 等：修改左操作数
- 自增自减 ++ --：修改操作数
- 函数调用 ()：可能修改全局变量或静态变量
- 逗号运算符 ,：两侧表达式可能产生副作用
### 8.左值与右值
#### 哪些表达式可以做左值？
- 变量名：x
- 数组元素：arr[5]
- 结构体成员：p.x、p->y
- 指针解引用：*ptr
#### 哪些表达式只能做右值？
- 字面量（除字符串外）：123、3.14、'a'
- 算术表达式：a + b
- 关系/逻辑表达式：a > b、a && b
- 函数调用返回值（非左值返回时）
- 强制类型转换结果：(int)3.14
### 9.常量表达式与整数常量表达式 
#### 哪些地方必须用常量表达式？
- 全局变量初始化：int g = 10;
- 静态局部变量初始化：static int s = 5;
- 数组大小（C99 前）：int arr[10];
- 枚举常量的值：enum { A = 1, B = 2 };
- 位域宽度：struct { int flag : 3; };
- case 标签：case 1:
#### 哪些地方必须用整数常量表达式？
- 数组大小（C99 前必须编译期确定）
- 位域宽度
- 枚举常量的值
- case 标签
- 静态断言 _Static_assert 的第一个参数
